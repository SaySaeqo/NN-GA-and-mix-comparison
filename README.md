# ProjektBadawczy
Algorytmy genetyczne z nielosową mutacją ukierunkowaną na
rozszerzenie przestrzeni poszukiwań. Poprawianie wydajności w
porównaniu np. z algorytmem gradientowym.

# Stylizacja
Funkcje,zmienne -> snake_case  
klasy -> camelCase  
stale -> TAB
